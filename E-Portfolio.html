  • Netflix Clone

    HTML • CSS • JavaScript • React • Vite • Firebase


    A Netflix-style streaming platform built with Firebase, featuring user authentication, real-time database integration, and dynamic movie/TV show browsing powered by an API. This project highlights my ability to recreate a polished, responsive UI while implementing secure login, data handling, and seamless content delivery


  • Amazon Clone – Fullstack E-Commerce Experience

    Next.js 14 • Firebase Auth & Firestore • Stripe Checkout • Tailwind CSS


    A recruiter-grade Amazon clone built with a modern fullstack stack. Features include secure authentication with Firebase, product browsing powered by Firestore, a persistent shopping cart, and real payments through Stripe Checkout with webhook handling. Designed with responsive layouts and reusable components for scalability and a professional e-commerce experience.


  • CryptoPlatform – Advanced Cryptocurrency Analytics & Portfolio Management

    Next.js • TypeScript • Tailwind CSS • Firebase • Web3


    A full-stack cryptocurrency platform that lets users track, analyze, and manage digital assets in real-time. Built with Web3 integration for wallet connectivity, Firebase authentication for secure logins, and Firestore for data persistence. Includes advanced portfolio tracking, trending market data, DeFi insights, and personalized dashboards — all wrapped in a modern, responsive UI powered by Tailwind CSS and TypeScript.


  • DeepSeek Clone – AI Chat Platform with OpenRouter Fallback

    Next.js 15 • React • Clerk Auth • Tailwind CSS • OpenRouter API


    A modern AI chat application inspired by DeepSeek. Built with Next.js 15 and Clerk authentication for secure user management, this project integrates the official DeepSeek API with OpenRouter as a fallback for reliability. Features include real-time conversations, context-aware memory, responsive design with Tailwind CSS, and a modular architecture (hooks, context, models) designed for scalability and clean maintainable code.


  • YouTube Clone – Video Streaming Platform

    React • Vite • JavaScript • CSS


    A fast and responsive YouTube clone built with React and Vite. Features include dynamic video rendering, channel-style layouts, interactive previews, and a streamlined user interface styled with CSS. Optimized for performance with Vite HMR, this project highlights my ability to build scalable front-end applications with modern tooling.
